透過光画像を使ったIIIFアプリケーション

※ 透過光画像との比較を容易にするため、元画像を上下反転させています。

IIIF Image API level 0を採用することで、本アプリケーションを画像を含めて静的ファイルのみで構築しています。

IIIF Image API level 0の画像生成には、 iiif_static を利用しています。

Image Comparison with a Slider

Digiratiが開発した画像比較ビューアを使用しています。

Mirador

画面左ペインの「Layers」タブを選択し、「opacity」を調整してください。

IIIF Curation Viewer

CODHが開発したIIIF Curation Viewer Embededを使用しています。
画面右上のラジオボタンを使用して、表示画像を選択してください。

Image Annotator

神崎正英氏が開発しているImage Annotatorを使用しています。
画面右上のchoiceを使用して、表示画像を選択してください。
